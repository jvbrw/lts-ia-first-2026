# APPENDIX: Design Notes

## Slide Count
- **Opening**: 6 slides
- **Block 1**: 25 slides
- **Block 2**: 25 slides
- **Closing**: 5 slides
- **Total**: ~61 slides

## Pacing Notes
- Data slides: let numbers land (2-3 seconds each)
- Quote slides: need breath room
- Case studies: can move faster
- Principles: slower, conceptual
- Decisions: urgency in delivery

## Visual Consistency
- Use the same treatment for all "principle" cards
- Use the same treatment for all "case study" cards (success vs failure differentiated)
- Comparison slides always same structure (DE → PARA)
- Numbers always prominent when they appear

## Possible Animations
- Reveal on quote slides (question, then answer)
- Data points can build sequentially
- Flow diagrams can animate left-to-right
- Keep animations minimal — professional pace

## Typography Hierarchy
1. **Section titles**: Largest, bold
2. **Slide headlines**: Large, medium weight
3. **Key statements**: Medium, can be light weight
4. **Supporting text**: Smaller, regular weight
5. **Labels/sources**: Smallest, muted color
